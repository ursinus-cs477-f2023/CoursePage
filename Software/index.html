Software Environment

NOTE: If you have previously installed anaconda python 3.9 or 3.10 for another course like CS 271 or CS 372, you will be fine to use that and do not need to install anything new.

We will be using a distribution of python known as Anaconda Python, which comes shipped with the majority of the tools that we need for the course. Click here to download Anaconda. Scroll to the bottom of the page until you see this (they've buried it...you should not have to sign up for an anaconda cloud account)

Choose the 64-bit version of Python 3.11 for your operating system (Windows/Mac/Linux), download it, and follow the default options during installation, including registering this as the default Python for your system.

Once you install Anaconda, you should be able to launch Spyder and Jupyter, which are the two environments we will be using to write and run code. Under Windows, you can launch these programs by searching for them in the search bar. Under Mac/Linux, you can run them by typing spyder and jupyter, respectively, in the command prompt.
